# Project 3 - online quiz

## Project Brief

<p>This is the Animal Quiz website revolved around animal nature. The website introduces the quiz game as well as providing a page with useful information about the website. You can access the website with this [link].</p>

## Features

### The website consists of three web pages with features accessible to the user.

- On the main page the screenshot below show the heading greeting users to the webite with accessible navigation links. The header is also fixed to the top of the page so it moves when the user scroll up or down.

![Heading](/assets/heading.png 'This is the heading of the website')

- A brief description is provided on the main page to help guide the user.

![brief description](/assets/brief-description.png)

- The footer of the page at the bottome is the contact details of the creators of the quiz if the user wish to get in contact.

![contact information](/assets/contact-details.png 'footer')

### How To Page

- The how to page is a dedicated page with clear instructions for users on how to take the quiz. This page guides users on starting the quiz, navigate through questions, and help them understand how their answers will be evaluated.

![howto page](/assets/howto-page.png)

- The "click on this link" provided is a link to a animal website which have a variety of information about many animals and nature to help the user learn more about animals. Therefor user's may gain more knowledge to complete questions correctly in the animal quiz game.

![animal website link](/assets/link.png)

- "Back to homepage" link is provided bottom left corner of the page to return to main page.

![back button](/assets/back-button.png)

### Animal Quiz Page

- When the quiz is initialised from the homepage "Animal Quiz" navigation link, the first question is displayed randomly using multiple choice options for the user to choose from.

![The Quiz](/assets/the-quiz.png)

- When the user select their answer they can submit it using the next button and go to the next question which will be displayed with different multiple choice options again.

![Next Button](/assets/next-button.png)

- Once user has completed the quiz, they will receive a result of their total score.

![Results](/assets/results.png)

## Technologies Used

- HTML was used as the fundamental of the website
- CSS was used for the styling of the website
- Javascript was used to create various elements of the website and used for the quiz functionality.
- Visual Studio Code is the IDE used to create the Animal Quiz website.
- Git was used for the version control of the website
- GitHub was used to host the code of the website
