About/How To Quiz

The animal quiz game is a platform provided to test the user knowledge about animals & enthisize the user's view on animals. Many people around the globe are animal lovers.

  1. To take the quiz users need to have some animal knowledge before starting the quiz. Please Click on this link to learn more about any animal. Hopefully enough information is provided for the user to be prepared for the challenging questions.


  2. Once user is ready to take the quiz, the user can navigate back to the home page using the link "Back to Home Page" below this page.


  3. On the home page there is a "Start Quiz" on the navigation bar on top of the web page. The user may click this link to start the quiz.


  4. The user may take their time to work through the quiz questions as there are no time limits. The aim of this quiz is to give the user more insight of an animal.


  5. The user will receive their final score upon completing the quiz.
